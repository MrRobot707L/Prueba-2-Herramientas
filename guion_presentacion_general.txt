NARRATIVA DEL PROYECTO (2-3 minutos)
Quien presenta: Antonia Montecinos
--------------------------------------------------------------------------------

El problema que abordamos es el siguiente: una vina necesita clasificar sus 
vinos antes de la cata final. Tradicionalmente esto requiere expertos 
sommeliers, lo cual es costoso y subjetivo.

Nuestra solucion propone usar las propiedades quimicas medibles del vino, como 
acidez, pH, nivel de alcohol y sulfatos, para predecir automaticamente si un 
vino alcanzara calidad premium.

Utilizamos el dataset Wine Quality Red de UCI Machine Learning Repository, que 
contiene 1,599 muestras de vino tinto portugues con 11 variables quimicas y una 
puntuacion de calidad del 3 al 8.

Para nuestro modelo, transformamos el problema en clasificacion binaria: los 
vinos con calidad 7 o superior se clasifican como Premium, y el resto como 
Estandar.

MODULO DE ANALISIS DE DATOS (3-4 minutos)
Quien presenta: Ignacio Ramirez
--------------------------------------------------------------------------------

El modulo de Analisis Exploratorio de Datos que yo desarrolle permite visualizar 
y entender el dataset antes de aplicar machine learning.

Primero mostramos las estadisticas descriptivas del dataset: medias, 
desviaciones estandar, valores minimos y maximos de cada variable quimica.

Luego visualizamos la distribucion de la variable objetivo. Aqui podemos ver que 
la mayoria de los vinos tienen calidad 5 o 6, mientras que los vinos premium con 
calidad 7 u 8 son menos frecuentes. Esto representa un problema de clases 
desbalanceadas.

En la seccion de Transformacion de Datos mostramos como convertimos el problema 
a clasificacion binaria. Creamos la columna target donde 1 significa Premium y 0 
significa Estandar.

Finalmente, el mapa de correlacion nos permite identificar que variables 
quimicas influyen mas en la calidad. Por ejemplo, el alcohol tiene correlacion 
positiva con la calidad, mientras que la acidez volatil tiene correlacion 
negativa.

Respecto a porque usamos formato pkl en vez de csv: el formato pickle es binario 
y permite cargar los datos mucho mas rapido, ademas de preservar exactamente los 
tipos de datos sin necesidad de parsear texto cada vez.

MODULO DE PREDICCION (3-4 minutos)
Quien presenta: Cristian Vergara
--------------------------------------------------------------------------------

El modulo de prediccion implementa una red neuronal MLP, que significa 
Perceptron Multicapa.

La arquitectura del modelo tiene:
- Una capa de entrada con 11 neuronas, una por cada variable quimica
- Una capa oculta con 64 neuronas y activacion ReLU
- Una segunda capa oculta con 32 neuronas
- Una capa de salida con 1 neurona y activacion Sigmoid para clasificacion 
  binaria

El usuario puede configurar los hiperparametros desde la interfaz:
- Numero de epocas de entrenamiento
- Tasa de aprendizaje
- Proporcion de datos para test

Durante el entrenamiento mostramos una barra de progreso y al finalizar 
graficamos la curva de aprendizaje que muestra como disminuye el error a lo 
largo de las epocas.

Finalmente, el usuario puede ingresar valores quimicos de un vino nuevo y el 
modelo predice si sera Premium o Estandar.
