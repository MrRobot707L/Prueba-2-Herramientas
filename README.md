# Prueba2Herramientas

Proyecto de ejemplo en Python.

## Requisitos

- Python 3.x
- (Opcional) Entorno virtual `.venv`

## Instalación

1. Activar el entorno virtual (si existe):

```powershell
& .\.venv\Scripts\Activate.ps1
```

2. Instalar dependencias (si se añaden en el futuro):

```powershell
pip install -r requirements.txt
```

## Uso

Ejecutar el archivo `main.py`:

```powershell
python .\main.py
```

O ejecutar `app.py`:

```powershell
python .\app.py
```
